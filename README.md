# Flex-dashboard_Angular20

A modern, flexible, and customizable dashboard platform built with Angular 20. This project demonstrates advanced Angular features including standalone components, signals, and Material UI integration.

## 🌟 Features

- **Customizable Widgets**: Add, resize, and configure widgets dynamically
- **Modern UI**: Built with Angular Material and custom styling
- **Responsive Design**: Adapts to different screen sizes
- **Standalone Components**: Uses Angular 20's standalone architecture
- **State Management**: Implements signal-based state management
- **Grid Layout**: Dynamic grid system for widget positioning

## 🚀 Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm (v9 or higher)
- Angular CLI (v20.2.0 or higher)

### Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/flex-dashboard.git
```

2. Navigate to project directory

```bash
cd flex-dashboard
```

3. Install dependencies

```bash
npm install
```

4. Start the development server

```bash
npm start
```

5. Open your browser and navigate to `http://localhost:4200`
